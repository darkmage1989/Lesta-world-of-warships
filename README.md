Тестовое Задание для Lesta-Games

Для создания использовался шаблон cra-template-typescript

Используемые библиотеки:
React,
Redux,
redux toolkit,
Mui,
typescript
Apollo,

Доступные скрипты - 
npm run start 
npm run build

Перед запуском проекта, необходимо установить зависимости npm i либо npm install
